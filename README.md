wndwj
=====

chrome插件，定制屏蔽


安装
------------
 
   -  点这里下载[wndwj.zip](https://github.com/xingzw/wndwj/archive/master.zip)
   -  解压缩
   -  按图1.2.3.指示安装
   -  ![image](http://i.imgur.com/QyeWxya.png)

使用
----------

   -  浏览帖子内容时自动生效。
   -  点击“wndwj”按钮，可隐藏或显示屏蔽内容。
   -  ![image](http://i.imgur.com/4clVnWi.png?1)
